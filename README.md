## MQ5 / SEN0130 — Φυσικό Αέριο / LPG

### Τεχνικά Χαρακτηριστικά
| Παράμετρος | Τιμή |
|---|---|
| Τύπος εξόδου | Αναλογική τάση |
| Εύρος εξόδου | 0 – 5 V |
| Εύρος ανίχνευσης LPG | 200 – 10.000 ppm |
| Load resistor (RL) | 10 kΩ (στη μονάδα DFRobot) |
| Τροφοδοσία | 5 V |
| Χρόνος προθέρμανσης | >24 ώρες |

### Ανάλυση Datasheet

#### Βήμα 1 — Rs από τάση
```
Rs = 10 kΩ × (5,0 - Vout) ÷ Vout
```

#### Βήμα 2 — R0 σε καθαρό αέρα
Από το datasheet: `Rs/R0 = 6,5` σε καθαρό αέρα.

```
R0 = Rs_καθαρός_αέρας ÷ 6,5
```

#### Βήμα 3 — Μετατροπή σε ppm (καμπύλη LPG από datasheet)
Εξίσωση καμπύλης για **LPG**:

```
ppm = 1000 × (Rs/R0) ^ (1 ÷ -2,138)
```

Οι σταθερές `1000` και `-2,138` προκύπτουν από **log-log γραμμική παλινδρόμηση**
(linear regression) στα σημεία της καμπύλης του datasheet.

### Πλήρης Τύπος
```cpp
float v     = analogRead(pin) * (5.0 / 1023.0);
float Rs    = 10.0 * (5.0 - v) / v;          // kΩ
float ratio = Rs / R0;
float ppm   = 1000.0 * pow(ratio, 1.0 / -2.138);
```

### Απλοποιημένος Τύπος (για μαθητές)
```
ppm ≈ analogRead × 10
```

### ⚠️ Σημαντικό
- Χρόνος προθέρμανσης **>24 ώρες**
- Απαιτείται βαθμονόμηση R0 σε καθαρό αέρα
- Ο MQ5 αντιδρά επίσης σε CO και αλκοόλη
